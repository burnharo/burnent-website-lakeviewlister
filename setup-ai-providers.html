Setup / AI Provider Keys

Setup guide

Get your AI provider keys.

Lakeview Lister currently exposes OpenAI, Anthropic, Google Gemini, and Moonshot in the Profile screen. You only need one provider to get started, but you can add more than one and test them inside the app.

Same pattern every time

  1. Create or sign in to the provider account.
  2. Open the provider’s API keys or developer settings area.
  3. Create the key and copy it immediately if the portal only shows it once.
  4. Paste it into Lakeview Lister Profile and click Test Key.

Where keys go

Use the Profile screen inside Lakeview Lister. The app stores these keys locally in the macOS Keychain.

Before you start

Start simple

You only need one provider to begin. Once one key is connected and tested, Lakeview Lister is ready for Draft Builder and Fast ID.

OpenAI

  1. Sign in to the OpenAI Developer Platform.
  2. Open the API key area for the project you want to use.
  3. Create a new secret key and copy it to a safe place.
  4. Paste it into OpenAI API Key in Lakeview Lister, then click Test Key.

If you organize work by project, use a dedicated key for Lakeview Lister so usage and permissions stay separated.

OpenAI quickstart · OpenAI key permissions

Anthropic

  1. Sign in to the Anthropic Console.
  2. Open your settings and go to the API keys area.
  3. Create a key in the workspace you want to use for Lakeview Lister.
  4. Paste it into Anthropic API Key in the app and run Test Key.

Anthropic’s docs describe API access through the web Console and note that keys are managed from the Console settings area.

Anthropic API overview · Anthropic Console

Google Gemini

  1. Sign in to Google AI Studio.
  2. Open the Gemini API key page.
  3. Create a new key or copy an existing one assigned to your account.
  4. Paste it into Google API Key in Lakeview Lister and click Test Key.

Google’s Gemini docs explicitly route API key creation through Google AI Studio.

Gemini API keys · Gemini quickstart

Moonshot / Kimi

  1. Sign in to the Moonshot Open Platform.
  2. Open the console API keys page.
  3. Create an API key in the default project and copy it.
  4. Paste it into Moonshot API Key in Lakeview Lister and run Test Key.

Moonshot’s official Kimi API docs point users to the Open Platform console API keys page for key creation.

Moonshot Open Platform · Moonshot docs mentioning API keys

Inside Lakeview Lister

What to do after you copy a key

Open Profile, paste the key into the matching provider field, save if needed, then use the app’s built-in Test Key action before running Draft Builder or Fast ID.
